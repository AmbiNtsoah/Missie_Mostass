# 🎙️ Missie Moustass

> Application desktop Java permettant d'enregistrer et de chiffrer des messages audio pour une communication sécurisée.

---

## 📌 Description

**Missie Moustass** est une application desktop développée en Java dans le cadre d'un projet pédagogique. Elle permet à un utilisateur d'enregistrer des fichiers audio et de les chiffrer, garantissant ainsi que seul le destinataire prévu peut les lire et les écouter.

Ce projet a été conçu pour explorer et mettre en pratique plusieurs concepts fondamentaux du développement logiciel :

- 🧹 **Clean Code** — écriture d'un code lisible, maintenable et bien structuré
- 🔐 **Chiffrement** — sécurisation des fichiers audio via des algorithmes de chiffrement
- ⚙️ **CI/CD** — automatisation des tests et du déploiement avec des outils d'intégration et de livraison continues

---

## 🎯 Objectif

L'objectif principal de Missie Moustass est de permettre l'échange de messages audio sécurisés entre utilisateurs. Une fois un message enregistré, il est chiffré et ne peut être déchiffré que par la personne destinataire, assurant ainsi la confidentialité des communications.
